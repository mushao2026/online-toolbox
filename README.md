# 在线工具箱

免费实用的在线工具集合，所有工具均在浏览器本地运行，数据不上传服务器。

## 包含工具

1. 🖼️ 图片压缩 - 支持 JPG/PNG/WebP
2. { } JSON格式化 - 格式化、压缩、验证
3. 📱 二维码生成 - 自定义样式
4. 🎨 颜色转换 - HEX/RGB/HSL
5. 📝 文字统计 - 字数、阅读时间
6. 🔐 Base64编解码 - 文本和图片
7. 🔗 URL编解码 - 编码解码转换
8. ⏰ 时间戳转换 - Unix时间戳
9. .* 正则测试 - 实时匹配高亮
10. 🔑 密码生成 - 安全随机密码
11. 📊 文本对比 - 差异高亮
12. 📄 Markdown预览 - 实时渲染

## 特点

- ✅ 纯前端实现，无需后端
- ✅ 数据本地处理，保护隐私
- ✅ 响应式设计，支持手机
- ✅ 无需注册，完全免费

## 部署方式

### 方式一：Vercel（推荐）

1. 访问 [vercel.com](https://vercel.com) 注册账号
2. 安装 Vercel CLI：`npm i -g vercel`
3. 在项目目录运行：`vercel`
4. 按提示完成部署

### 方式二：Netlify

1. 访问 [netlify.com](https://netlify.com) 注册账号
2. 直接拖拽项目文件夹到 Netlify

### 方式三：GitHub Pages

1. 创建 GitHub 仓库
2. 上传代码
3. 在 Settings → Pages 启用
